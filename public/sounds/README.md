# Notification Sounds

This directory contains audio files for real-time notifications.

## Required Files

- `notification.mp3` - Sound played when a new order is received in the kitchen dashboard

## Adding Sounds

1. Add your notification sound file as `notification.mp3`
2. Recommended format: MP3, 44.1kHz, 128kbps
3. Keep file size under 100KB for fast loading
4. Duration: 1-2 seconds recommended

## Free Sound Resources

- [Freesound.org](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)
- [Notification Sounds](https://notificationsounds.com/)
